获取TOKEN

在调用各接口前，都要先获取TOKEN

接口地址

https://server:port/oauth/token

请求方法

POST

请求头(header)

参数名
类型
必填
参数位置
描述
值
Authorization
string
是
header
访问令牌，
授权clientId和clientSecret请联系业务人员
将clientId和clientSecret拼接，中间加冒号，再用base64工具编码。
比如clientId为client，clientSecret为123qwe，拼接后为:client:123qwe，使用base64编码后为:Y2xpZW50OjEyM3F3ZQ== ，最后再拼接上Basic，中间加空格。
结果为：Basic Y2xpZW50OjEyM3F3ZQ==

请求body

参数名
类型
必填
参数位置
描述
grant_type
string
是
bodyParam
授权模式为密码模式，固定传 password
username
string
是
bodyParam
用户名
password
string
是
bodyParam
用户密码

请求示例

$ curl -X POST \
    -H "Accept: application/json" \
    -H "Authorization: Basic Y2xpZW50OjEyM3F3ZQ==" \
    -H "Content-Type: application/x-www-form-urlencoded" \
    -d "grant_type=password" \
    -d "username=admin" \
    -d "password=123qwe" \
    http://server:port/api/oauth/token
