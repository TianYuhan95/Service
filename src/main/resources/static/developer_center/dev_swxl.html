声纹训练

接口调用时要求上传的语音文件有效时长大于30秒。

接口地址

https://server:port/api/voice_print/train

请求方法

Multi-part POST

请求body

参数名
类型
必填
参数位置
描述

string
是
bodyParam

remark
JSON

bodyParam
JSON字符串，包括样本人名name，图片picture等信息
files
audio/wav
是
bodyParam
语音样本，可上传多个文件

请求示例

$ curl -X POST \
    -H "Accept: application/json" \
    -H "Authorization: bearer Access_Token" \
    -H "Content-Type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW" \
    -F "dirId=1" \
    -F "remark=XXX" \
    -F "files=@test/data/test01.wav" \
    -F "files=@test/data/test02.wav" \
    https://server:port/api/voice_print/train
