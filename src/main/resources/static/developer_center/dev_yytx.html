语音听写

基于WebSocket协议的语音听写接口，适用于不间断实时语音数据转写场景。

接口地址

wss://server:port/api/online_decoder

接口参数

content-type=audio/x-raw,+layout=(string)interleaved,+rate=(int)8000,+format=(string)S16LE,+channels=(int)1

参数说明：

参数
描述
audio/x-raw
音频原始数据
+rate=(int)16000
采样速率（HZ）8000
+layout=(string)interleaved
单通道
+format=(string)S16LE:16bit
16bit little-endian格式
+channels=(int)1
频道范围1

JSON格式结果定义

{"status":0,"result":{"hypotheses":[{"transcript":"识别结果文本"}],"final":false}}

参数说明：

Key
类型
描述
status
int
处理状态 — 0：成功、1：无语音(输入语音包括大量空音或静默音)、2：终止(识别过程存在异常)、9：无法获取资源(系统负载达到上限)
hypotheses
list
预测结果
transcript
string
识别结果
final
boolean
每句话最终预测结果时为true，否则为false

JSON格式结果示例

{"status":9}
{"status":0,"result":{"hypotheses":[{"transcript":"see on"}],"final":false}}
{"status":0,"result":{"hypotheses":[{"transcript":"see on teine lause."}],"final":true}}

结束标记

由于超时（如长时间没有语音输入）原因，引擎会返回结束标记（字符串“EOS”），表示当前会话已经结束，调用方如果还需要继续使用引擎接口，需要再次发起新的会话。
